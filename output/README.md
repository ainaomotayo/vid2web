# Nexus Analytics Website

A professional landing page for a data analytics firm.

## Files
- `index.html`: The structure of the landing page.
- `styles.css`: The styling including responsive design and animations.
- `scripts.js`: Interactive elements like smooth scrolling and scroll-reveal effects.

## Features
- Responsive design for mobile, tablet, and desktop.
- Intersection Observer for scroll-triggered animations.
- Modern CSS variables for easy theme customization.
- Semantic HTML5 for accessibility and SEO.
